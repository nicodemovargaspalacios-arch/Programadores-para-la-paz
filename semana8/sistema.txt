whoami: Muestra el nombre de usuario actual con el que se esta ejecutando la terminal de Termux. La funcion general es indicar qué usuario del sistema operativo está ejecutando las acciones.Además ayuda a saber bajo qué perfil de usuario se guardarán las configuraciones globales (~/.gitconfig) o si tienes los permisos adecuados para acceder a las carpetas locales de tus repositorios.Nota en Termux: En Android, Termux.
uname -a: Muestra toda la información del sistema operativo, el kernel de Linux y la arquitectura del hardware del dispositivo. La funcion general es imprimir el nombre del kernel, la versión, la fecha de compilación y el tipo de procesador (como aarch64 o armv7l). Pero en cuanto a su funcionalidad con GIT es fundamental para diagnosticar fallas de compatibilidad. Si intentas instalar herramientas adicionales, extensiones de Git (como Git LFS) o credenciales seguras (como git-credential-libsecret), necesitas conocer la arquitectura exacta que te da uname -a para descargar el binario correcto para tu teléfono.
Conocer la información del sistema es fundamental para garantizar la estabilidad, seguridad y rendimiento de un servidor. Al administrar infraestructura, estos datos te permiten tomar decisiones proactivas antes de que ocurra una falla crítica.
